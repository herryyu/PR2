\chapter{序言}

\begin{flushleft}
    Rust是一门为系统编程而生的语言。
    \mkpar
    然而在如今，绝大多数正在工作的程序员对于系统编程并不熟悉，那么便需要对其进行一些解释了。至今它仍是我们做任何事的基础。
    \mkpar
    你合上了笔记本电脑。操作系统会感知到，并中止所有正在运行的程序，熄灭屏幕，最后让电脑进入睡眠。过了一会儿，你又打开了
    电脑，屏幕与其它组件再次被启动，并且每个程序均能从它中断的地方恢复过来，我们认为这是理所当然的事。但这都是系统程序员
    写了大量代码而得到的成果。
    \mkpar
    系统编程一般被用到这些地方：
    \begin{itemize}
        \item 操作系统
        \item 各种设备驱动
        \item 文件系统
        \item 数据库
        \item 运行在廉价设备、亦或是极度稳定的设备上的代码
        \item 密码学
        \item 媒体解码器（负责读取及写入音频、视频或者图像文件的软件）
        \item 媒体处理（例如，语音识别或照片编辑软件）
        \item 内存管理（例如，实现一个垃圾回收器）
        \item 文本渲染（将文本和字体转变为像素）
        \item 实现高级编程语言（比如 JavaScript 和 Python）
        \item 网络
        \item 虚拟化技术以及软件容器
        \item 科学模拟
        \item 游戏
    \end{itemize}
    简单来说，系统编程是一种资源受限的程序设计，它面向每个字节（byte）和每个CPU周期（cycle）来编程。
    \mkpar
    涉及支持一个基础应用程序运行的系统代码的数量会相当惊人。

    \mkbaresec{适合阅读本书的人群}
    如果你已是一位系统程序员并且准备好迎接C++的替代品，那么此书适合你。若你是对于任何编程语言均有经验的开发者，无论是
    C\#，Java，Python，JavaScript或者是其它东西，那么此书也适合你。
    \mkpar
    但是，你不能光学习Rust。要想最大化地利用该门语言，你还需获得一些系统编程的经验。我们建议你在阅读此书的同时用Rust
    实现一些有关系统编程的项目。试着去做一些你从来没有构建过的，能利用好Rust的迅捷、并发以及安全性的东西。对于本序言开
    端列出的这些议题应该能为你带来一些灵感。

    \mkbaresec{我们编写本书的原因}
    在我们初期钻研Rust的时候便开始着手编写这本我们期望中的书了。我们的目标是开门见山地，将Rust重要且新的概念整理出来，
    进而清晰而深入地表述它们，以便减少“通过试错来学习”这样的现象。

    \mkbaresec{本书导航}
    本书的前两章介绍了Rust，并在我们移步至\boldred{第三章}的基础数据类型前提供了一段简短的总览。有关所有权和引用的概念
    位于第\boldred{四、五}章节。我们建议按顺序通读本书前五章。
    \mkpar
    第\boldred{六}至第\boldred{十}章涵盖了该门语言的基础知识：表达式（\boldred{第六章}）、错误处理（\boldred{第七
    章}）、crates和modules（\boldred{第八章}）、结构体（\boldred{第九章}）以及枚举型和模式（\boldred{第十章}）。
    对于这些部分快速阅览一下没问题，但一定别跳过错误处理这章，相信我们。
    \mkpar
    第十一章涵盖了traits和generics，也是最后两个你需要了解的重要概念。Traits类似于Java或C\#里的interfaces。它们同
    时也是使Rust能支持你将你自定义的类型整合进该语言本身的主要方式。\boldred{第十二章}展示了traits是如何支持运算符重
    载的，另外\boldred{第十三章}涵盖了更多的多功能traits。
    \mkpar
    了解traits和generics之后便可解锁本书的余下部分。闭包（Clousures）和迭代器（iterators），这两个你绝对不想错过的
    强大工具，各自被涵盖于第\boldred{十四、十五}章。对于剩余章节你可以按自由顺序进行阅读，或者根据需求对它们进行深度挖掘。
    这些章节涵盖了该门语言的剩余部分：集合（collections）（\boldred{第十六章}）、字符串和text（\boldred{第十七章}）、
    输入输出（\boldred{第十八章}）、并发（\boldred{第十九章}）、异步编码（\boldred{第二十章}）、宏（\boldred{第二十
    一章}）、unsafe编码（\boldred{第二十二章}）以及在其它语言中调用函数（\boldred{第二十三章}）。
